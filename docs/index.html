tg-streaming-bot

A self-hosted Telegram bot that streams music & video straight into your group's voice chat.

View on GitHub Deploy your own
Built with Pyrogram · py-tgcalls · ffmpeg · yt-dlp

Everything you'd want in a voice-chat player

One bot + a dedicated assistant account join the call and stream — you control it all from chat.

🎵

YouTube & files

Play from a search query, a direct URL, or any audio file posted in the chat.

🎬

Video & live

Stream videos and live sources (m3u8, YouTube-live) with selectable 720/480/360 quality.

🔎

Search picker

/search lists the top results as buttons — pick one to play as audio or video.

📻

Internet radio

Dozens of built-in stations with the live now-playing track shown on the video feed.

📼

Recording

Record the radio or a video stream to a voice/video message — stop anytime, with a live tracklist.

📁

Local library

Browse and play your own media folders, with audio-track and subtitle selection at play time.

🎛️

Full controls

Pause, resume, skip, seek, volume, loop, shuffle, queue — plus an inline control panel.

⚡

Hardware transcoding

VAAPI GPU or CPU transcoding handles HEVC/MKV and anything the stream engine can't take raw.

🔁

Self-healing

Auto-reconnects after a drop, resumes the last track after a restart, and leaves when idle.

Commands

A few of the essentials — full list in the bot via /help.

Play

  • /play <query|url> — music
  • /vplay <query|url> — video
  • /vstream <link> — live / m3u8
  • /search <query> — results picker
  • /radio — internet radio
  • /library · /lplay — local media

Control

  • /pause /resume /skip /stop
  • /seek 12:30 — jump to a time
  • /volume 1-200
  • /loop /shuffle /clear
  • /info — now playing + controls

Capture & setup

  • /record — record audio/video
  • /stoprec — stop & send
  • /song · /video — download
  • /userbotjoin · /reload
  • /alive · /ping

Quick start

Runs anywhere Docker does. You'll need a bot token, Telegram API credentials, and a session string for the assistant account.

  1. Clone and configure: 
    git clone https://github.com/antlis/tg-streaming-bot.git
cd tg-streaming-bot
cp example.env .env   # fill in the Required block
  2. Build & run: 
    docker compose up -d --build
  3. In your group: add the bot as admin, start a voice chat, then /userbotjoin and /play <song>.

Full setup, configuration reference, and notes on running where Telegram is filtered are in the README.
